Instalátor aplikací 1.0

Spusť položku "Instalátor aplikací 1.0" (Můžeš jako správce)

Napiš název aplikace, kterou chceš nainstalovat

  • avast - Antivirus firmy Avast (stáhne se i s tím prohlížeč "Avast Secure Browser")
  • chrome - Prohlížeč "Google Chrome"
  • handbrake - Aplikace HandBrake, která slouží k ztrátové kompresi videa.
  • malwarebytes - Antimalware Malwarebytes
  • avg - Antivirus AVG
  • python - Programovací jazyk Python
  • visualstudiocode - Profesionální textový editor "Visual studio code", k programování apod.
  • visualstudio - Kompilátor (Hlavně na C++) (Něco podobného jako Visual studio code)
  • notepad++ - Textový editor na programování "Notepad++"
  • windows10 - Media creation tool k stáhnutí Windows 10 iso, nebo k upgradování/downgradování počítače
  • windows11 - Media creation tool k stáhnutí Windows 11 iso, nebo k upgradování počítače
  • java - Programovací jazyk "Java"
  • spotify - Aplikace na poslouchání hudby "Spotify"
  • roblox - Videohra "Roblox"
  • msteams - Aplikace na volání a chatování "Microsoft Teams"
  • avira - Antivirus "Avira"
  • msoffice2007 - Sada balíčku "Microsoft Office 2007"
  • 7zip - Tvořič zip souborů a rozbalovač "7Zip"
  • winrar - Tvořič rar a zip souborů a rozbalovač "WinRar"
  • netdesktopruntime60 - Přídavek od Microsoftu pro funkci různých aplikací (např. HandBrake) "NerDesktopRuntime 6.0"
  • obs - Nahrávání obrazovky, a aplikace na streamování "OBS" (Open Brodcast Software)
  • steam - Aplikace pro stahování videoher "Steam"
  • windirstat - Grafické znázornění souborů a složek na disku "WinDirStat"
  • brave - Webový prohlížeč "Brave"
  • firefox - Webový prohlížeč "Opera"
  • fileshredder - Aplikace pro trvalé odstranění věcí, nebo vyčištění volného místa
  • ccleaner - Aplikace na vyčištění počítače "CCleaner" (který osobně nedoporučuju)
  • recuva - Aplikace na obnovení trvale odstraněných souborů "Recuva"
  • processhacker - Pokročilý správce úloh, který dokáže ukončit i systémový proces "Process Hacker"
  • crystaldiskinfo - Aplikace na sledování stavu disku "Crystal Disk Info"
  • processexplorer - Velmi pokročilý správce úloh "Process Explorer"
  • coretemp - Aplikace na sledování teplory procesoru "Core Temp"
  • autoruns - Aplikace se seznamem všech služeb a aplikací, které spouští po startu "AutoRuns"
  • tcpview - Aplikace ke kontrolování, jak jiné aplikace komunikují se sítí "TCPView"

Až se vám instalační soubor stáhne, tak stiskněte enter ke spuštění instalačního programu
